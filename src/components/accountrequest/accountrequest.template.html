Adicionar novo cliente

A adição de novos clientes funciona apenas por meio de convites, portanto preencha os dados abaixo para que seja gerado um código de acesso, e um endereço no qual o cliente poderá realizar seu cadastro.

O identificador irá lhe auxiliar a localizar o convite na tabela de convites, portanto, atribua um nome de fácil indentificaçao, por exemplo: "Convite de John Locke".
PS. Este campo só é visível por você.
